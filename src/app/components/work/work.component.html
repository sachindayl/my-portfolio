Duckworth Lewis Calculator

An application developed for the sport of cricket to calculate the final total in the case of an interruption (eg: rain). This is a personal project which is maintained in Github. User interface testing is done using Espresso. Continuous Integration is done using Circle CI.


The D/L method was devised by two British statisticians, Frank Duckworth and Tony Lewis.

Emertext

A semester course project done to learn mobile application development. This application assists the user to request help from others. It periodically sends text messages with a link to the current location. This will happen until the phone is switched off or the user cancels the request.



Ardina

The final semester project developed at The Ohio State University. This application was developed from scratch to connect doctors and patients without them having to travel to the hospital. It uses a Firebase (NoSQL) database for patient and doctor registration. Once doctors show their availability in their portal, patients can select them and a call is created between them.
